# WorkIt Web App

Full-stack web application built with:

- **Frontend**: Vue.js + Tailwind CSS (Vite)
- **Backend**: NestJS
- **Database**: PostgreSQL

## Structure

- `workit-frontend/`: Vue 3 app using Vite and Tailwind
- `workit-backend/`: NestJS app with RESTful API
- `database.sql`: SQL schema to initialize the PostgreSQL database
- `workit_schema.dbml`: Optional DBML diagram (for database visualization)

## Setup Instructions

Each folder has its own README and setup instructions.
